# Structure Viewer

It's an extension for VSCode to preview material structures directly.

It supports:

- ".xyz" or ".extxyz"
- ".cif"

Next, more formats will be supported:

- ".in" (CONQUEST)
- ".gjf" (Gaussian View)
- ......
